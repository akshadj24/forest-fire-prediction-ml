Project Overview

The Forest Fire Prediction project is designed to evaluate meteorological data and predict the Fire Weather Index (FWI). The FWI is a crucial metric used globally to estimate the danger of forest fires.

The Dataset

This application is trained on the Algerian Forest Fires Dataset. It comprises data from two regions in Algeria (Bejaia and Sidi Bel-abbes) over a period from June to September.

Model Architecture

The underlying machine learning model utilizes Ridge Regression. Ridge regression is a technique for analyzing multiple regression data that suffer from multicollinearity. By adding a degree of bias to the regression estimates, ridge regression reduces the standard errors.

Key Input Features

  • Temperature (Temp): temperature in Celsius.
  • Relative Humidity (RH): relative humidity in %.
  • Wind Speed (WS): wind speed in km/h.
  • Fine Fuel Moisture Code (FFMC): represents moisture of litter and other cured fine fuels.
  • Duff Moisture Code (DMC): represents moisture content of loosely compacted organic layers.
  • Drought Code (DC): represents moisture of deep, compact organic layers.
  • Initial Spread Index (ISI): expected rate of fire spread.
  • Buildup Index (BUI): amount of fuel available for combustion.
