# 🔥 Forest Fire Prediction using Machine Learning

## 📌 Overview
This project predicts the likelihood and severity of forest fires using environmental and weather-related data. The system uses regression-based machine learning models and is deployed using Flask for real-time prediction.

## 📊 Dataset
- Algerian Forest Fires Dataset
- Features include temperature, humidity, wind speed, rain, and fire weather indices.

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Flask
- HTML/CSS
- Pickle

## 🧠 Machine Learning Models
- Linear Regression
- Ridge Regression (final model)
- Feature Scaling using StandardScaler

## 📈 Model Evaluation
- Metrics used: MAE, RMSE, R² Score
- Ridge Regression selected based on better generalization performance.

## 🚀 Application
- Flask-based web application
- User inputs environmental parameters
- Model predicts fire risk value in real time
